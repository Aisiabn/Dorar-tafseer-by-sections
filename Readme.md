# dorar-tafseer-scraper

سكربت Python لاستخراج **موسوعة التفسير** من [dorar.net/tafseer](https://dorar.net/tafseer) وإعادة تنظيمها حسب نوع القسم.

## الفكرة

بدلاً من حفظ كل سورة في ملف واحد، يجمع السكربت المحتوى **حسب نوع القسم** عبر القرآن كاملاً:

```
dorar_by_section/
  فهرس.md
  غريب_الكلمات.md        ← غريب الكلمات من الفاتحة حتى الناس
  بلاغة_الايات.md        ← بلاغة الآيات من كل السور
  الفوائد_التفسيرية.md
  ...
```

## المميزات

- الآيات القرآنية بين ﴿ ﴾
- الأحاديث النبوية بين « »
- حواشي مرقّمة ومفصولة أسفل كل موضع
- ترقيم تسلسلي فريد للحواشي في كل ملف (لا تداخل)
- دمج ذكي للعناوين المتشابهة (يتجاهل الحركات والفروق البسيطة)
- تعريف كل سورة مضمّن ضمن قسمه المناسب

## المتطلبات

```bash
pip install requests beautifulsoup4
```

## الاستخدام

```bash
python dorar_tafseer_by_section.py
```

للاختبار على عدد محدود من السور، عدّل في أعلى الملف:

```python
TEST_SURAHS = 3   # أول 3 سور فقط
```

لجلب القرآن كاملاً:

```python
TEST_SURAHS = None
```

## الناتج

ملف Markdown لكل قسم + `فهرس.md` يضم كل الأقسام مع عدد مواضعها.

الملفات متوافقة مع **Obsidian** و**Pandoc** (لتحويلها إلى Word أو PDF مع حواشي حقيقية).
